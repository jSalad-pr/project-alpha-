# Project Alpha

> A self-hosted infrastructure lab built to develop practical experience in Linux administration, networking, Docker, and systems engineering.

---

## Overview

Project Alpha is my personal homelab where I design, deploy, and maintain self-hosted infrastructure using industry-standard tools.

The goal of this project is to gain hands-on experience with technologies commonly found in enterprise IT, cloud infrastructure, and modern data centers.

---

- Ubuntu Server
- Docker

- AdGuard Home
