# Services

This document describes the services currently deployed in Project Alpha.

---

## Portainer

**Purpose**

Provides centralized management of Docker containers.

**Responsibilities**

- Container deployment
- Container monitoring
- Stack management

---

## Vaultwarden

**Purpose**

Self-hosted password manager.

**Responsibilities**

- Secure credential storage
- Multi-device password synchronization

---

## Nginx Proxy Manager

**Purpose**

Reverse proxy for internal services.

**Responsibilities**

- HTTPS termination
- Reverse proxy routing
- SSL certificate management

---

## AdGuard Home

**Purpose**

DNS filtering and network-wide ad blocking.

**Responsibilities**

- Internal DNS
- DNS filtering
- Network visibility

---

## WireGuard

**Purpose**

Secure remote VPN access.

**Responsibilities**

- Remote administration
- Encrypted network access

---

## Uptime Kuma

**Purpose**

Infrastructure monitoring.

**Responsibilities**

- Service health checks
- Availability monitoring
- Status dashboard

---

## Smallstep CA

**Purpose**

Private Certificate Authority.

**Responsibilities**

- Internal TLS certificates
- PKI management
